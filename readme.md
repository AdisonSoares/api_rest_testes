# Usuário api rest

Essa api tem a função de cadastrar usuários, porém seu objetivo principal é a prática de testes unitários
com JUnit e Mockito.

Tecnologias e métodos utilizados:
- 
* Java 17
* Spring boot
* Banco H2
* Boas práticas
* Testes com JUnit e Mockito
